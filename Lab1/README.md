# SYSC3010 Lab1 Deliverables# Lab1

## Files
- **showmyname.py** - Displays name scrolling on SenseHAT LED
- **showsensordata.py** - Displays temperature, pressure, humidity on SenseHAT LED
- **ifconfig_screenshot.png** - RPi network configuration (MAC and IP address)

## How to Run
```bash
python3 showmyname.py
python3 showsensordata.py
```

## Requirements
- Raspberry Pi with SenseHAT
- Python 3 with SenseHAT library
